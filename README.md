## Introduction
Telegram bot <a href="https://t.me/nearseedbot">@nearseedbot</a> helps you to correct seed phrase with misprints

### Installation
```
$ npm install
```

### Usage
```
$ node bot.js
```

Send to the bot your wallet address and seed phrase with misprints and bot will try to fix it
